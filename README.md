# Facture Simple

React + Tailwind CSS + Font Awesome scaffold.

## Get started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## Structure

- `src/`
  - `components/`
  - `layout/`
  - `pages/`
  - `utils/`
  - `hooks/`
  - `assets/`
  - `styles/`
